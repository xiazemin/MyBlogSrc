Go Search Extension是一款可以在浏览器地址栏快速、方便、高效地搜索 Go 文档、第三方库的浏览器插件，支持 Chrome/Firefox/Edge 。类似于 Rust 社区的 Rust Search Extension。

下载地址：

Chrome Web Store

Firefox

Microsoft Edge

主要功能：

1.搜索 Go 标准库文档

可通过关键词搜索 package 、func 、interface，快速查看其定义和描述，也可选中并跳转到文档页面查看具体说明。
